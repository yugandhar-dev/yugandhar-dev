## 👋 About me

I build flexible, high-performance web applications and scalable mobile experiences. With over 8.5 years of global software engineering experience across Australia and India, I specialize in modern JavaScript architectures, full-stack development, and cross-platform mobile solutions.

- 🔭 Currently building **nyburbs** — a local services marketplace (NestJS + Expo + Supabase/PostgreSQL)

- 📍 Melbourne, Australia · Open to senior frontend / full stack roles
